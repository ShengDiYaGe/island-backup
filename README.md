Island Backup
==============
以 `aiohttp` `asyncio` 为主编写的备份程序  

备份匿名版上的串, 图片到本地, 以html形式查看  


使用方法
-------
需要python >= 3.5

`python3 main.py`  

输入 url , 目前只至此 h.nimingban.com 的串  

```shell
url
http://h.nimingban.com/t/6048436?r=6048436
```

内容在backup文件夹里面.
